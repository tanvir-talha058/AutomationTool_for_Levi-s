## Creating Your Own Template

1. **Open Excel/LibreOffice Calc**
2. **Create columns** with clear headers
3. **Add your data** in rows below headers
4. **Save as .xlsx** format
5. **Test with 3-5 rows** first

## Important Notes

- First row should contain column headers
- Avoid special characters in headers
- Keep data clean (no formulas in data cells)
- Save as .xlsx or .xls format
- Close the file before uploading to the tool

## Column Selection

After uploading, you'll be able to select which column to use for automation:
- Select the column that contains the data you want to submit to the website
- For example, if automating searches, select the "Search_Term" column

## Best Practices

✅ **DO**:
- Use clear, descriptive column names
- Test with small datasets first
- Keep data simple and clean
- Save backup copies

❌ **DON'T**:
- Use merged cells
- Include empty rows between data
- Use complex formulas
- Leave file open while uploading
